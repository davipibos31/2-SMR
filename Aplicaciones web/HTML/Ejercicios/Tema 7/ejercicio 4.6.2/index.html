Linux commands
Commands Description
passwd Changes your user password:
  1. Type your old password
  2. Insert new password
  3. . Conrm new password
- User’s home directory (short-cut for:
/home/username)
ls Lists folders and les in current directory
mkdir Create new directory into the current one:
mkdir newdir
cd Changes directory: cd test (go to a directory named ‘test’)
cd .. (go to parent directory)
cd ~ (go to home directory)
rm Removes specied le or directory:
rm filename (remove single lename)
rm *.txt (remove ALL .txt les into current directory)
rm -r dirname(remove directory and its les)

Please be careful when you use the -f option!
rmdir Removes specied EMPTY directory
rmdir dirname
pwd Prints current absolute path
man Shows specied command’s manual page:
man ls (shows ls help)		 
vi x.sh
 VI is a text editor. If x.sh does not exists, vi creates a new le called x.sh and opens it; otherwise, it just opens the existing le. 
less textfile
 lesss is a text pager. Opens (read only) textle le. You can use up and down arrows to move across the text, shares many commands with VI.
chmod Changes POSIX permissions of a le or directory. Allows to protect les from unwanted access:
r : read permission
w : write permission
x : exec permission

chmod + x file.sh (allows execution)
chmod - w file.sh (denies write)
top Changes owner of a le or directory:
chown username file.sh
cat Prints the content of a le on screen
grep Filters specied text le, and shows the lines that contains the pattern:
grep pattern file.sh
You can use also pipe the output of another command:
cat file.sh | grep home
cat file.sh | grep "home page"
